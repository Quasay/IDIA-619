HW 4: Product Page

Create a Web page with the following things:
- Text input field for Product Name
- Text input field for Year Manufactured
- Button to add Product to a list (array)
- Button to View List

When the user clicks the add product button, the system checks to see if the product was made before 2008. If it was, it can be added to the list. If it was not, the user gets a message saying it is too new and can't be added. If the user clicks the button to view the list, the products are listed in the page.
